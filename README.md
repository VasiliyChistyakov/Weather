# AppWeather

Приложение AppWeather показывает данные о погоде введеного города [OpenWeather API](https://openweathermap.org/api), сохраняет название города при повторной загрузке приложения.

- CAGradientLayer
- URLSession
- MVVM
- JSON
- UISearchBar
- User Defaults
